# PDF Düzeltici (Neon Motoru)

PDF belgelerindeki yazım hatalarını yapay zeka ile tespit eden ve orijinal font/hizalamayı bozmadan düzelten Python tabanlı bir araçtır.

## Kurulum
1. Projeyi bilgisayarınıza klonlayın: git clone https://github.com/hikmetozddemir/pdf-duzeltici.git
2. Kütüphaneleri kurun: pip install -r requirements.txt
3. Bir .env dosyası oluşturun ve içine GOOGLE_API_KEY=anahtarınız yazın.
4. Çalıştırın: streamlit run app.py
